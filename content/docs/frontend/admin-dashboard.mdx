# Admin Dashboard

System administration dashboard for managing SmartFall infrastructure.

## Key Features

### System Statistics
- Total users
- Active devices
- Falls today
- System health status

### User Management
- List all users
- Create/edit users
- Change user roles
- Deactivate users

### Device Management
- View all devices
- Enable/disable devices
- View device logs
- Manage firmware updates

### Fall Management
- View all fall events
- Filter by severity
- Investigate suspicious events
- Generate reports

### System Logs
- Activity logs
- Error logs
- Device logs
- User access logs

## Management Functions

### User Actions
```
Create User
Edit User
Reset Password
Change Role
Deactivate
Delete
```

### Device Actions
```
View Status
View Logs
Disable Device
Enable Device
Update Firmware
View Telemetry
```

### System Actions
```
View Statistics
Export Data
View Logs
Run Maintenance
Generate Reports
```
